# Project Guide

## Project Title

Splunk-Driven Security Monitoring for MQTT Brokers

## Objective

This project demonstrates a lightweight Security Information and Event Management (SIEM) solution for monitoring MQTT broker activities using Splunk Enterprise.

## Technologies

- Splunk Enterprise
- Mosquitto MQTT Broker
- MQTT
- Kali Linux
- Shell Scripting
- Linux

## Attack Scenarios

- Brute-force login attack
- Topic flooding attack
- Unauthorized publishing
- Session hijacking

## Detection

The project monitors Mosquitto broker logs and uses Splunk search queries to detect malicious behavior in real time.

## Outcome

The system successfully detects multiple MQTT attack scenarios and visualizes alerts through Splunk dashboards.
