# Splunk-Driven Security Monitoring for MQTT Brokers

## Overview

This project presents a lightweight Security Information and Event Management (SIEM) framework for monitoring MQTT brokers in IoT environments using Splunk Enterprise and the Mosquitto MQTT broker.

The framework detects multiple MQTT-based attacks in real time by analyzing broker log events and generating security alerts through Splunk dashboards.

---

## Features

- Real-time MQTT monitoring
- Brute-force attack detection
- Topic flooding detection
- Unauthorized publish detection
- Session hijacking detection
- Splunk dashboards and alerts
- Lightweight deployment suitable for IoT environments

---

## Technologies Used

- Splunk Enterprise
- Mosquitto MQTT Broker

- Kali Linux
